Introduction

EtherComputer - my new project. It is device that exchanges data through ether(air).

How does it work?

In fact it is a radio transciever working in sepecified band. Such devices("clients") are exchhaging packages of data by special protocol(TODO).

EtherPCDataTransferProtol(EPDTP)

To make it universal i choose Unicode as encoding of text. Controller "translates" message to binary code, next puts firmware id, time, etc. and sends it to the air.
